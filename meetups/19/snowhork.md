## 今日の成果
ベーシック圏論 p.36まで（自然変換の導入）

圏の定義を確認

圏とは：
「対象と射」の集まり
＋射同士が合成できて
＋合成が結合法則を満たして
＋各対象に対して恒等射が存在する

合成関数は結合法則を満たすことの確認
```rb
class Proc
  def c(f2)
    -> (x) { self.call(f2.call(x)) }
  end
end

f = -> (x) { x+1 }
g = -> (x) { 3*x }
h = -> (x) { x**2 }

f.c(g).c(h).call(5)
# (3*(x+1))**2
# => 76
f.c(g.c(h)).call(5)
# (3*(x**2))+1
# => 76
```
